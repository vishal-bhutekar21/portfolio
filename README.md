## ğŸ¨ Customization

### Update Theme Colors

Edit CSS variables in `style.css`:

```css
:root {
    --primary: #2563eb;
    --accent: #059669;
    --bg: #f8fafc;
    /* ... more variables */
}
```

### Update Content

1. **Personal Info**: Edit `index.html` sections
2. **Projects**: Update project cards in the Projects section
3. **Images**: Replace images in `/images` folder
4. **Resume**: Replace `assets/vishal_cv.pdf`
5. **Certifications**: Add PDFs to `assets/certificates/`

### Add/Remove Sections

Sections are modular - simply add/remove `<section>` blocks in `index.html`
